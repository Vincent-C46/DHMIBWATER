# 컨트롤 스타일 사용 가이드

## 1. 스타일 시스템 개요

프로젝트의 모든 스타일은 `DHBIMWATER.UI/Styles` 폴더에 정의되어 있으며, `Generic.xaml`을 통해 통합 관리됩니다.

### 스타일 파일 구조
- `Colors.xaml` - 색상 정의 (Primary, Gray, Success, Warning, Error, Info 등)
- `TextBlocks.xaml` - TextBlock 스타일
- `Buttons.xaml` - Button 스타일
- `TextBoxes.xaml` - TextBox 스타일
- `CheckBoxes.xaml` - CheckBox 스타일
- `ComboBoxes.xaml` - ComboBox 스타일
- `ListBoxes.xaml` - ListBox 스타일
- `DataGrids.xaml` - DataGrid 스타일
- `GroupBox.xaml` - GroupBox 스타일
- `TabControls.xaml` - TabControl 스타일
- `ProgressBars.xaml` - ProgressBar 스타일

---

## 2. XAML에서 스타일 리소스 참조하기

### 2.1. Window.Resources에 Generic.xaml 추가

모든 View에서 스타일을 사용하려면 다음과 같이 리소스 딕셔너리를 추가해야 합니다:

```xml
<Window.Resources>
    <ResourceDictionary>
        <ResourceDictionary.MergedDictionaries>
            <ResourceDictionary Source="/DHBIMWATER.UI;component/Styles/Generic.xaml"/>
        </ResourceDictionary.MergedDictionaries>
    </ResourceDictionary>
</Window.Resources>
```

## 3. TextBlock 스타일 사용법

### 3.1. 제목 및 본문 스타일

```xml
<!-- 대제목 -->
<TextBlock Text="제목" Style="{StaticResource HeadingTextBlockStyle}"/>

<!-- 부제목 -->
<TextBlock Text="부제목" Style="{StaticResource SubHeadingTextBlockStyle}"/>

<!-- 본문 -->
<TextBlock Text="본문 내용" Style="{StaticResource BodyTextBlockStyle}"/>

<!-- 캡션 (작은 글씨) -->
<TextBlock Text="캡션 내용" Style="{StaticResource CaptionTextBlockStyle}"/>

<!-- 라벨 -->
<TextBlock Text="라벨:" Style="{StaticResource LabelTextBlockStyle}"/>
```

### 3.2. 상태 표시 스타일

```xml
<!-- 성공 메시지 -->
<TextBlock Text="성공적으로 완료되었습니다" Style="{StaticResource SuccessTextBlockStyle}"/>

<!-- 경고 메시지 -->
<TextBlock Text="주의가 필요합니다" Style="{StaticResource WarningTextBlockStyle}"/>

<!-- 에러 메시지 -->
<TextBlock Text="오류가 발생했습니다" Style="{StaticResource ErrorTextBlockStyle}"/>

<!-- 정보 메시지 -->
<TextBlock Text="추가 정보입니다" Style="{StaticResource InfoTextBlockStyle}"/>
```

### 3.3. 강조 스타일

```xml
<!-- Primary 색상 강조 -->
<TextBlock Text="중요한 텍스트" Style="{StaticResource PrimaryTextBlockStyle}"/>

<!-- 굵은 글씨 -->
<TextBlock Text="굵은 텍스트" Style="{StaticResource BoldTextBlockStyle}"/>

<!-- 연한 글씨 -->
<TextBlock Text="연한 텍스트" Style="{StaticResource LightTextBlockStyle}"/>
```

## 4. Input 컨트롤 스타일 사용법

### 4.1. Button

```xml
<Button Style="{StaticResource PrimaryButtonStyle}"
        Content="확인"
        Width="100"
        Command="{Binding SaveCommand}"/>
```

**특징:**
- Primary 색상 배경
- 호버 시 색상 변경
- 클릭 시 투명도 효과
- 둥근 모서리 (CornerRadius: 8)

### 4.2. TextBox

```xml
<TextBox Style="{StaticResource PrimaryTextBoxStyle}"
         Text="{Binding InputText}"
         Width="250"
         Padding="8"/>
```

**특징:**
- 포커스 시 Primary 색상 테두리
- 둥근 모서리 (CornerRadius: 6)
- 기본 여백 5px

### 4.3. CheckBox

```xml
<CheckBox Style="{StaticResource PrimaryCheckBoxStyle}"
          Content="동의합니다"
          IsChecked="{Binding IsAgreed}"/>
```

**특징:**
- 체크 시 Primary 색상 표시
- 둥근 모서리 체크박스
- 체크 마크 애니메이션

## 5. Selection 컨트롤 스타일 사용법

### 5.1. ComboBox

```xml
<ComboBox Style="{StaticResource PrimaryComboBoxStyle}"
          Width="250"
          SelectedIndex="0"
          ItemsSource="{Binding Options}">
    <ComboBoxItem Content="옵션 1"/>
    <ComboBoxItem Content="옵션 2"/>
    <ComboBoxItem Content="옵션 3"/>
</ComboBox>
```

**특징:**
- 드롭다운 버튼 포함
- 호버 시 Primary 색상 테두리
- 둥근 모서리 (CornerRadius: 6)
- 드롭다운 최대 높이 200px

### 5.2. ListBox

```xml
<ListBox Style="{StaticResource PrimaryListBoxStyle}"
         Width="250"
         Height="200"
         ItemsSource="{Binding Items}">
    <ListBoxItem Content="항목 1"/>
    <ListBoxItem Content="항목 2"/>
    <ListBoxItem Content="항목 3"/>
</ListBox>
```

**특징:**
- 선택 시 Primary 색상 배경
- 호버 시 연한 배경 표시
- 둥근 모서리 (CornerRadius: 10)
- 각 항목 둥근 모서리 (CornerRadius: 6)

## 6. Progress 컨트롤 스타일 사용법

### 6.1. ProgressBar

```xml
<ProgressBar Style="{StaticResource PrimaryProgressBarStyle}"
             Width="300"
             Height="8"
             Value="{Binding Progress}"
             Maximum="100"/>
```

**특징:**
- Primary 색상 진행 표시
- 둥근 모서리 (CornerRadius: 4)
- 기본 높이 8px
- 배경 색상 자동 설정

## 7. Container 컨트롤 스타일 사용법

### 7.1. GroupBox

```xml
<GroupBox Style="{StaticResource PrimaryGroupBox}"
          Header="그룹 제목"
          Width="400">
    <StackPanel>
        <TextBlock Text="내용 1" Style="{StaticResource BodyTextBlockStyle}"/>
        <TextBlock Text="내용 2" Style="{StaticResource BodyTextBlockStyle}"/>
    </StackPanel>
</GroupBox>
```

**특징:**
- Primary 색상 헤더
- 둥근 모서리 테두리 (CornerRadius: 10)
- 헤더 배경 흰색
- 내부 여백 15px

### 7.2. TabControl

```xml
<TabControl Style="{StaticResource PrimaryTabControlStyle}"
            Width="600"
            Height="300">
    <TabItem Header="탭 1">
        <StackPanel Margin="20">
            <TextBlock Text="탭 1 내용"/>
        </StackPanel>
    </TabItem>
    <TabItem Header="탭 2">
        <StackPanel Margin="20">
            <TextBlock Text="탭 2 내용"/>
        </StackPanel>
    </TabItem>
</TabControl>
```

**특징:**
- 선택된 탭 하단에 Primary 색상 바
- 호버 시 Primary 색상 텍스트
- 선택된 탭 굵은 글씨
- 투명 배경

## 8. Data 컨트롤 스타일 사용법

### 8.1. DataGrid

```xml
<DataGrid Style="{StaticResource PrimaryDataGridStyle}"
          Width="700"
          Height="300"
          ItemsSource="{Binding DataList}">
    <DataGrid.Columns>
        <DataGridTextColumn Header="이름" Binding="{Binding Name}" Width="*"/>
        <DataGridTextColumn Header="설명" Binding="{Binding Description}" Width="2*"/>
        <DataGridTextColumn Header="상태" Binding="{Binding Status}" Width="*"/>
    </DataGrid.Columns>
</DataGrid>
```

**특징:**
- 헤더 스타일 자동 적용 (PrimaryDataGridHeaderStyle)
- 선택된 행 Primary 색상 텍스트
- 호버 시 연한 배경
- 교대 행 색상 자동 적용
- 둥근 모서리 (CornerRadius: 10)
- 그리드 라인 숨김

## 12. 주의사항

1. **반드시 Generic.xaml을 참조해야 합니다**
   - Window.Resources에 Generic.xaml을 추가하지 않으면 스타일을 찾을 수 없습니다

2. **StaticResource 사용**
   - 모든 스타일은 `{StaticResource 스타일명}` 형식으로 참조합니다
   - DynamicResource는 사용하지 않습니다

3. **일관성 유지**
   - 프로젝트 전체에서 동일한 스타일을 사용하여 UI 일관성을 유지합니다
   - 커스텀 스타일이 필요한 경우 기존 스타일을 BasedOn으로 상속받아 사용합니다

4. **네이밍 규칙**
   - Primary로 시작하는 스타일이 기본 스타일입니다
   - 예: PrimaryButtonStyle, PrimaryTextBoxStyle

5. **색상 직접 지정 금지**
   - 하드코딩된 색상 대신 Colors.xaml의 리소스를 사용합니다
   - 예: `Background="#2196F3"` ❌ → `Background="{StaticResource PrimaryBrush}"` ✅
